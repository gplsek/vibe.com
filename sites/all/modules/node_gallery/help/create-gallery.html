There are two workflows available to you to start creating galleries and adding images. Please note that the second workflow requires that you have the plupload module installed. Which one you choose is a matter of personal taste, try them both and see which you prefer.

  • Create a new empty gallery first, then upload images.

    • Visit node/add/node-gallery-gallery and fill out the node form as you would for any other node.
    • When you create a new empty gallery, you will be sent directly to the 'Upload New Images' tab on the newly created node. If you have plupload installed (trust us, you should), you'll be able to upload a batch of images all at once.
    • After uploading images, you can click on the 'View' tab to view thumbnails of your images, or you can visit the 'Manage Images' tab to modify titles, etc. Lastly, if you wish to apply a custom sort to this gallery, you may do so from the 'Sort Images' tab.

  • Upload a batch of images, then create a gallery to store them in.
